# 通识课

## 选课要求

### 学分要求

通识课程每门为 2 学分。以软件学院 2024 级为例，需要从以下类别中各选择一门课程：

- **TY**
- **TD**
- **TS**
- **TR**
- **TH**

总计需要完成 **10 学分**的通识课程。

### 选课注意事项

1. **仔细阅读学院文件**：选课前务必查阅学院发布的通识课选课文件，了解具体要求，避免盲目选课。

2. **避免超选**：超出要求的学分需要额外缴费，收费标准为每学分 80 元。选错课程可能导致需要补选，既增加学习负担，又产生额外费用。

3. **选课策略**：建议根据个人兴趣和课程难度合理选择，不要仅凭课程名称随意选择。

## 学习建议

### 绩点管理

通识课程可以用于提升整体绩点。如果对绩点有要求，建议：

- **线下课程**：积极参与课堂讨论和互动，提高平时成绩
- **线上课程**：可以使用刷课脚本（适用于大学生劳动教育、就业指导等通用课程）节省时间，但需注意课程要求

### 时间管理

如果对绩点要求不高，以通过考试为目标：

- 可以参考学习通签到情况合理安排出勤
- 将更多时间投入到专业课程的学习中

### 学习态度

通识课程的主要目的是完成毕业学分要求。建议：

- 不要投入过多时间在通识课上
- 将主要精力放在专业课程的学习上
- 合理平衡通识课与专业课的学习时间

## 重要提醒

不同学院、不同年级的通识课要求可能存在差异，请务必以学院官方发布的最新文件为准。
