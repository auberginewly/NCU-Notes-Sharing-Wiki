# 虚拟定位

虚拟定位工具可以用于修改设备的 GPS 位置信息，适用于需要位置验证的场景。

## iOS 平台

### GeoPort

GeoPort 是一款开源的 iOS 虚拟定位工具，支持通过数据线或 Wi-Fi 连接进行定位模拟。

#### 系统要求

- 需要 iPhone 开启开发者模式
- 支持数据线连接或 Wi-Fi 连接

#### 项目资源

- **GitHub 项目地址**：https://github.com/davesc63/GeoPort
- **官方网站**：https://geoport.me/

### 爱思助手

爱思助手是一款功能丰富的 iOS 设备管理工具，包含虚拟定位功能。

#### 项目资源

- **官方网站**：https://www.i4.cn/
- **使用教程**：https://zhuanlan.zhihu.com/p/696772055

### 使用说明

1. 根据选择的工具访问相应网站
2. 下载并安装工具
3. 按照教程进行配置
4. 注意遵守相关法律法规和学校规定
