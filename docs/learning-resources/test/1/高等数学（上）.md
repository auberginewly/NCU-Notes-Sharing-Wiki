# 高等数学（上）

## 学习资料

### 学习路线

- **速成学习**：高等数学（上）可以通过速成课程快速掌握，建议在观看速成课程后完成相应练习以巩固知识。
- **真题练习**：建议在期末考试前两周开始练习历年真题，题型相对固定，有助于熟悉考试形式。
- **证明题复习**：证明题（如泰勒展开、中值定理相关题目）可先跳过，留待最后统一复习。
