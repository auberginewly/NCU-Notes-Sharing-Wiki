# AI 润色提示词

在提交贡献之前，请使用以下提示词让 AI 帮助你润色和优化内容：

## 基础润色提示词

```
你是一位专业的学术文档编辑助手。请帮我润色以下内容，要求：

1. **表达清晰准确**：确保文字表达清晰，没有歧义，专业术语使用准确
2. **符合学术规范**：使用规范的学术语言，避免口语化表达
3. **易于理解**：内容要通俗易懂，适合大学生阅读
4. **风格统一**：保持与项目整体风格一致，使用简体中文
5. **结构优化**：如果内容结构混乱，请帮助优化段落结构和逻辑
6. **格式规范**：确保 Markdown 格式正确，标题层级清晰

请润色以下内容：

[在这里粘贴你需要润色的内容]
```

## 使用建议

1. **选择合适的提示词**：根据你要润色的内容类型，选择对应的提示词模板
2. **提供上下文**：如果可能，提供一些上下文信息，帮助 AI 更好地理解内容
3. **多次迭代**：如果第一次润色效果不理想，可以调整提示词或要求 AI 进一步优化
4. **人工检查**：AI 润色后，请务必人工检查一遍，确保内容准确无误
5. **保持原意**：润色时要注意保持原意，不要改变核心内容

3. 重点练习往年真题，熟悉考试题型和难度
