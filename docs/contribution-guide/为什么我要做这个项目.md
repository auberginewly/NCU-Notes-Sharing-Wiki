# 为什么我要做这个项目

## 初衷

在 NCU 学习的这些时光里，我深深体会到了找学习资料的困难。每次准备考试、做作业或者想要深入学习某个知识点时，总是要花大量时间在各种群聊、论坛、网盘里翻找资料。有时候找到了，却发现资料不完整、版本不对，或者链接已经失效。

更让我印象深刻的是，很多同学都在重复着同样的"找资料"过程。明明大家都有一些很好的学习资源，却因为缺乏一个统一的平台，导致资源分散、难以共享。

## 我想解决的问题

1. **资料分散**：学习资料散落在各种群聊、网盘、论坛中，难以集中管理
2. **信息不对称**：新生不知道哪些资料有用，哪些是过时的
3. **重复劳动**：每个人都在重复寻找同样的资料
4. **资源浪费**：很多优质的学习资源没有被充分利用

## 我的想法

我想创建一个**开放、共享、持续更新**的学习资料库，让 NCU 的同学们能够：

- 📚 **快速找到**需要的课程资料
- 🎯 **少走弯路**，直接使用经过验证的学习资源
- 🤝 **互相帮助**，共同完善这个知识库
- 💡 **传承经验**，让学弟学妹们受益

## 这个项目的价值

### 对学生

- 节省大量找资料的时间
- 获得经过整理和验证的学习资源
- 了解课程的学习路线和重点
- 参考往年真题，更好地准备考试

### 对社区

- 建立 NCU 学习资源的共享平台
- 促进同学之间的知识分享
- 形成良性的学习互助氛围
- 让知识传承下去

## 我的承诺

我会持续维护这个项目，不断更新和完善内容。但我也知道，一个人的力量是有限的，所以我希望：

- 有更多的同学能够参与进来
- 大家一起贡献、一起完善
- 让这个项目真正成为 NCU 学生的共同财富

## 未来展望

我希望这个项目能够：

1. **覆盖更多课程**：从大一到大四，从基础课到专业课
2. **内容更丰富**：不仅有资料，还有学习路线、经验分享
3. **社区更活跃**：有更多的贡献者，形成良性的协作氛围
4. **帮助更多人**：让每一届 NCU 学生都能从中受益

## 写在最后

这个项目不是为了炫耀，也不是为了什么功利的目的。我只是希望，通过我的努力，能够让 NCU 的同学们在学习路上少一些困难，多一些帮助。

如果你觉得这个项目有价值，欢迎你一起参与进来。无论是贡献一份资料、纠正一个错误，还是分享一条经验，都是对这个项目的支持。
