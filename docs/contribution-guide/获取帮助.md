# 获取帮助

如果你在使用这个项目时遇到问题，或者需要帮助，这里有几个途径可以获取支持。

## 常见问题

### 1. 找不到某个课程的资料？

- 检查一下课程名称是否正确
- 查看是否有对应的文件夹和文件
- 如果确实没有，欢迎你贡献相关资料！

### 2. 文件链接失效了？

- 可能是文件被移动或删除了
- 请在 GitHub 上提交 Issue 报告这个问题
- 我会尽快修复或更新链接

### 3. 想要添加新的学习资料？

- 查看 [贡献指南](./贡献指南.md) 了解如何贡献
- 按照规范提交 Pull Request
- 使用 [AI润色提示词](./AI润色提示词.md) 优化内容

### 4. 发现内容有错误？

- 可以直接提交 Issue 报告错误
- 或者 Fork 项目，修改后提交 Pull Request
- 你的纠错对项目很有价值！

### 5. 如何使用这个网站？

- 在导航栏选择对应的分类（应试资源、CS资源等）
- 在侧边栏浏览具体的课程
- 点击课程名称查看详细内容

## 如何提问

### 在 GitHub 上提交 Issue

如果你遇到问题或想要提出建议，可以在 GitHub 上提交 Issue：

1. 访问项目的 GitHub 仓库
2. 点击 "Issues" 标签
3. 点击 "New Issue"
4. 选择合适的问题类型
5. 详细描述你的问题或建议

### 提问时请包含：

- **问题描述**：清楚地说明你遇到了什么问题
- **期望结果**：你希望得到什么结果
- **实际结果**：实际发生了什么
- **相关截图**：如果有，请附上截图
- **环境信息**：浏览器、操作系统等（如果相关）

### 提问示例

**好的提问：**
```
问题：高等数学（上）页面的真题链接无法打开

期望：能够正常下载真题文件

实际：点击链接后显示 404 错误

截图：[附上截图]

环境：Chrome 浏览器，Windows 11
```

**不好的提问：**
```
链接坏了
```

## 联系方式

### GitHub Issues

这是**最推荐**的联系方式，因为：

- 问题可以被追踪和记录
- 其他人可能也有同样的问题
- 便于后续维护和更新

### 其他方式

如果你有紧急问题或需要私下联系，可以通过以下方式：

- **GitHub Discussions**：用于讨论和问答（如果项目启用了）
- **Pull Request**：如果你已经修复了问题，直接提交 PR

## 反馈和建议

我们非常欢迎你的反馈和建议！无论是：

- 功能建议
- 内容改进
- 用户体验优化
- 新功能需求

都可以通过 GitHub Issues 提交。你的每一个建议都可能让这个项目变得更好！

## 贡献代码

如果你想直接贡献代码或内容：

1. 查看 [贡献指南](./贡献指南.md)
2. Fork 项目
3. 创建分支进行修改
4. 提交 Pull Request

- 我会尽量在 **48 小时内**回复你的 Issue
- 如果是 Pull Request，我会尽快审查
- 如果问题比较复杂，可能需要更长时间

感谢你的耐心！
